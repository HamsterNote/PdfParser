# Task 8: Cover Serialization - Evidence

## Date: 2026-05-23

## Changes Made

### `resolveCoverAvailable` updated to handle `IntermediateImage`

**File:** `demo/demoDocumentSerialization.js`

The `resolveCoverAvailable` function now checks for `IntermediateImage` objects rather than raw strings. According to the 0.8.0 specification, `getCover()` returns `IntermediateImage | undefined`.

**Before:**
```javascript
const cover = await intermediate.getCover(0.25).catch(() => undefined)
const available = typeof cover === 'string' && cover.length > 0
```

**After:**
```javascript
const cover = await intermediate.getCover(0.25).catch(() => undefined)
const available = Boolean(
  cover &&
  typeof cover === 'object' &&
  'src' in cover &&
  typeof cover.src === 'string' &&
  cover.src.length > 0
)
```
